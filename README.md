# cld (cat linux detail)

`cld 是类似于neofetch的命令行工具(基于c语言)`

## Install

```
$ git clone https://github.com/LongYinStudio/cld.git
$ cd cld
$ make
$ <sudo> make install
```

## Uninstall

```
$ <sudo> make uninstall
```
